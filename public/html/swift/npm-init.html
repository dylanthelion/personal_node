To start a new node project (once npm and node are installed), enter 'npm init' into your Terminal. After you finish the wizard, a package.json file will be generated. This file will be referenced whenever you build your project. The following is an example of a package.json file. 

				
				
					{
			  			"name": "personal_site",
			 			"version": "1.0.0",
			  			"description": "Personal site, built on a MEAN stack",
			  			"main": "index.js",
			  			"scripts": {
			    			"test": "test"
			  			},
			  			"author": "dylanthelion",
			  			"license": "ISC"
					}
