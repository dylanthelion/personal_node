Sample package.json

To start a new node project (once node and npm are installed), enter 'npm init' into your Terminal/Bash. The following is an example of the generated package.json file, which will be referenced whenever you build your server. 

				
				
					{
			  			"name": "personal_site",
			 			"version": "1.0.0",
			  			"description": "Personal site, built on a MEAN stack",
			  			"main": "index.js",
			  			"scripts": {
			    			"test": "test"
			  			},
			  			"author": "dylanthelion",
			  			"license": "ISC"
					}
